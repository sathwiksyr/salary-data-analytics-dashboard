# Cloud-Based Salary Data Analytics Dashboard

This project analyzes salary trends using cloud data analytics tools.

## Tools Used
- Google BigQuery
- Looker Studio
- SQL

## Dashboard Features
- Average salary KPI
- Maximum salary KPI
- Salary vs experience scatter plot
- Salary distribution by experience range
- Data visualization charts

## Insights
Salary increases consistently with years of experience, with higher salary ranges observed for professionals with more than 8 years of experience.# salary-data-analytics-dashboard
